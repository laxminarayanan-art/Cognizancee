This program will allow the user to print the Nth fibonacci number.


Declare Integer i,r1,r2,temp,number
Output "Input the number"
Input number
If number==0 
      Output "0"
      If number==1
             Output "1"
             If number==2
                    Output "1"
             End
       End
Else 
       Assign r1=1
       Assign r2=1
       For i=1 to number-2
               Assign temp=r1+r2
               Assign r1=r2
               Assign r2=temp
       End
       Output "temp"
End
